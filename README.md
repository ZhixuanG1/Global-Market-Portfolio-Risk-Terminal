# Global Market & Portfolio Risk Terminal

An institutional-style financial risk analytics platform built with Python and Streamlit.

## Overview

This project provides an interactive market and portfolio risk terminal for multi-asset portfolio monitoring, risk measurement, scenario analysis, market surveillance, and financial data visualization.

## Key Features

- Portfolio performance and risk analytics
- Annualized return, volatility, Sharpe ratio, Sortino ratio and max drawdown
- Historical VaR, CVaR, parametric VaR and Cornish-Fisher VaR
- Stress testing and scenario analysis
- Monte Carlo simulation
- Efficient frontier and portfolio optimization diagnostics
- Factor exposure analysis
- Credit risk proxy analytics
- Liquidity risk analysis
- Interest-rate and yield-curve monitoring
- Global market monitor
- Equity and fixed income universe screener
- Yahoo Finance news feed integration

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- yFinance
